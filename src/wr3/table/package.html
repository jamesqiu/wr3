table 中包含生成各类报表的类；

AbstractTable   各类报表共用的一些处理；
FormTable       数据表单
ListTable       数据列表
RotateTable     旋转表
AggregateTable  聚合表
GroupTable      分组报表
FrameTable      框架报表
  -TopFrameTable
  -LeftFrameTable
CrossTable      交叉报表
CubeTable       多维表

CellFilter     接口，在process()中返回对Table/Row每一个Cell的处理。
PageBar        分页条Html，适用于 ListTable, RotateTable

各报表包含一个缺省的同名.ftl模板文件，可以用用户目录下的进行覆盖, common.ftl为公用部分。
