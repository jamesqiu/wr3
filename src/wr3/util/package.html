本package中的staitc方法调用, 可通过如下用法避免不必要冗余类名输入.:

import static wr3.util.Stringx.*;

	
Charsetx:     enc编码, 中文;
Classx:       动态类装载及方法调用;
CLI:          命令行参数处理;
Datetime:     Date,Time,Calendar的util;
Filex:        File,Directory,文本文件内容处理的util;
GroovyUtil:   Groovy,ConfigSlurp解析;
Json:         JSON组装;
Numberx:      int,double的util;
Regex:        JDK常用regex处理的util;
Stringx:      String处理的util;
Logx:         设置某些应用如Spring、hibernate的log级别;
Objectx:      进行Object的序列化和反序列化(使用Hessian2);
Rowx:         Row操作;
Tablex:       Table操作;
Servletx:     从Request得到各种url，session等;
Exec:         执行系统命令；
Annotationx:  处理类的Annotation注解；
