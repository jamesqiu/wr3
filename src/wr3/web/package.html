package概要说明:
  AppFilter          友好URL的Filter, 传给Appservlet
  AppServlet         根据controller,action,id等执行AppController
  AppController      控制器, 传入参数, 执行action, 返回值
  Params             表示所有request参数
  Session            表示所有session参数
  Render             返回json, html, text结果
  UriParser          从友好URL得到controller, action, id
  AclFilter          权限控制Filter		
  Head.ftl           Render#body()所用的head部分	

web处理流程：
request ------> AppFilter ------> UriParser | AppServlet ------> Render | AppController------> HotClass | Classx
WEB-INF/web.xml内容: 
<?xml version="1.0" encoding="ISO-8859-1"?>

<web-app xmlns="http://java.sun.com/xml/ns/javaee"
   xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
   xsi:schemaLocation="http://java.sun.com/xml/ns/javaee http://java.sun.com/xml/ns/javaee/web-app_2_5.xsd"
   version="2.5"> 

    <description>WebReport3</description>
    <display-name>WebReport3</display-name>

	<!-- AppFilter -->
	<filter>
		<filter-name>AppFilter</filter-name>
		<filter-class>wr3.web.AppFilter</filter-class>
		<init-param>
			 <param-name>wr3.home</param-name>
			 <param-value>e:/tomcat/webapps/wr3</param-value>
		 </init-param>
	</filter>
	<filter-mapping>
		<filter-name>AppFilter</filter-name>
		<url-pattern>/*</url-pattern>
	</filter-mapping>

	<!-- AppServlet -->
	<servlet>
		<servlet-name>wr3app</servlet-name>
		<servlet-class>wr3.web.AppServlet</servlet-class>
	</servlet>
	<servlet-mapping>
		<servlet-name>wr3app</servlet-name>
		<url-pattern>/app/*</url-pattern>
	</servlet-mapping>

</web-app>
